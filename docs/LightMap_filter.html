LightMap#filter

filter LightMap based on keys and values

Source:
  • LightMap.js, line 34

Example

const _ = new LightMap();
_.set( 'key', 'value' );
_.set( 'key1', 'value1' );

const result = _.filter(
    ( v, k ) => k === 'key'
);

// -> LightMap { 'key' => 'value' }
